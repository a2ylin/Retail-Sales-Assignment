# Retail Sales Assignment

## Project Overview

This project demonstrates basic data analysis using Python and pandas.

## Tech
- Python
- pandas
- matplotlib

## Tasks Completed
- Loaded the dataset
- Inspected the dataset
- Checked missing values
- Filled missing values
- Converted ORDERDATE to datetime
- Generated summary statistics
- Used value_counts() on key columns
- Created a bar chart of the top 5 countries by order count
- Exported the cleaned dataset as a new CSV file
